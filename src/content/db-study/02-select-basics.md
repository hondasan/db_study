# データの取得（SELECT文の基本）

SQLにおいて最も頻繁に使われるのが、データを取り出すための `SELECT` 文である。
この章では、データベースから狙ったデータを自由に取り出せるようになるための基本を解説する。

---

## SELECT文の基本構文

テーブルから全てのカラム（列）を取得する最もシンプルな構文は以下の通りである。

```sql
SELECT * FROM テーブル名;
```

*   `SELECT`: 「取得する」という命令。
*   `*` (アスタリスク): 「すべての列（カラム）」を意味する。
*   `FROM`: 「どのテーブルから取得するか」を指定する。
*   `;` (セミコロン): SQL文の終わりを示す。

### 特定のカラムだけを指定して取得する

すべての列ではなく、必要な列だけを指定して取得するのが実務では一般的である。列名をカンマ `,` で区切って指定する。

```sql
SELECT name, email FROM users;
```

## データを絞り込む（WHERE句）

特定の条件に合うレコード（行）だけを取得したい場合は、`WHERE` 句を使用する。

```sql
SELECT * FROM users WHERE age >= 20;
```

### よく使う比較演算子
*   `=` : 等しい
*   `!=` または `<>` : 等しくない
*   `>` / `<` : より大きい / より小さい
*   `>=` / `<=` : 以上 / 以下

### 複数の条件を組み合わせる (AND, OR)

```sql
-- 20歳以上、かつ、名前が「Alice」のユーザーを取得
SELECT * FROM users 
WHERE age >= 20 AND name = 'Alice';

-- 東京都在住、または、大阪府在住のユーザーを取得
SELECT * FROM users 
WHERE prefecture = '東京都' OR prefecture = '大阪府';
```

## データの並び替え（ORDER BY句）

取得したデータを特定の列を基準にして並び替えたい場合は、`ORDER BY` 句を使用する。

```sql
-- 年齢が若い順（昇順: ASC）に並び替えて取得
SELECT * FROM users ORDER BY age ASC;

-- 登録日が新しい順（降順: DESC）に並び替えて取得
SELECT * FROM users ORDER BY created_at DESC;
```

*   `ASC`: 昇順（Ascending）。省略した場合はデフォルトで昇順になる。
*   `DESC`: 降順（Descending）。大きい値から小さい値、新しい日付から古い日付の順になる。
