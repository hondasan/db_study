# データベースとSQLの基本

データベース（Database / DB）は、コンピュータシステム上で整理された**データの集まり**である。
現代のアプリケーション（Webサイト、スマホアプリ、基幹システムなど）のほぼすべてが、裏側でデータベースを利用してユーザー情報や投稿、ログなどのデータを管理している。

---

## リレーショナルデータベース（RDB）とは？

現在最も広く使われているのが**リレーショナルデータベース（RDB）**である。
データをエクセルのような「表（テーブル）」の形式で管理し、複数のテーブル同士を関連付ける（リレーション）ことができるのが特徴だ。

### RDBの構成要素
*   **テーブル (Table)**: データを格納する表全体。
*   **レコード (Record / 行)**: データの1件1件。
*   **カラム (Column / 列)**: データの項目（ID、名前、メールアドレスなど）。

<div class="note-box">
  <p><strong>コラム: データベース管理システム（DBMS）</strong></p>
  <p>データベース自体はデータの集まりだが、それを実際に管理・操作するソフトウェアをDBMS（Database Management System）と呼ぶ。代表的なRDBMS（リレーショナルDBMS）には、MySQL、PostgreSQL、SQLite、Oracle Databaseなどがある。</p>
</div>

---

## SQLとは？

SQL（Structured Query Language）は、リレーショナルデータベースを操作するための**標準言語**である。
データベースに対して「データを取得する」「データを保存する」「データを削除する」といった命令（クエリ）を送るために使用する。

例えば、ユーザー一覧のテーブルから名前が「Alice」のデータを取得したい場合、以下のようなSQLを記述する。

```sql
SELECT * FROM users WHERE name = 'Alice';
```
