# データベースとSQLの基本

データベース（Database / DB）は、コンピュータシステム上で整理された**データの集まり**です。
現代のアプリケーション（Webサイト、スマホアプリ、基幹システムなど）のほぼすべてが、裏側でデータベースを利用してユーザー情報や投稿、ログなどのデータを管理しています。

## リレーショナルデータベース（RDB）とは？

現在最も広く使われているのが**リレーショナルデータベース（RDB）**です。
データをエクセルのような「表（テーブル）」の形式で管理し、複数のテーブル同士を関連付ける（リレーション）ことができるのが特徴です。

### RDBの構成要素
*   **テーブル (Table)**: データを格納する表全体。
*   **レコード (Record / 行)**: データの1件1件。
*   **カラム (Column / 列)**: データの項目（ID、名前、メールアドレスなど）。

---

## SQLとは？

SQL（Structured Query Language）は、リレーショナルデータベースを操作するための**標準言語**です。
データベースに対して「データを取得して」「データを保存して」「データを削除して」といった命令（クエリ）を送るために使用します。

例えば、ユーザー一覧のテーブルから名前が「Alice」のデータを取得したい場合、以下のようなSQLを書きます。

```sql
SELECT * FROM users WHERE name = 'Alice';
```
